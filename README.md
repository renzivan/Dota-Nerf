# Dota 2 Profile Web App Using Vue.js

### API Source
[OpenDota API Documentation](https://docs.opendota.com)

### Getting started from scratch
```bash
# Install the Vue CLI
npm install -g @vue/cli

# New Vue Project
vue create hello-world

# Install vue-router
npm install vue-router

# Install axios
npm install axios

# Run application
npm run serve

```
